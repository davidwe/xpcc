#define MAIN_FUNCTION

Inhibits some stack operations at the beginning of main for avr-gcc. May save up a few bytes of stack memory.

Typical structure of an microcontroller program: 

 #include <xpcc/architecture/platform.hpp>
 
 MAIN_FUNCTION
 {
    ...
    
    while (1)
    {
        ...
    }
 }

#define ALWAYS_INLINE

Macro to force inlining on functions if needed. Compiling with -Os does not always inline them when declared only inline.
