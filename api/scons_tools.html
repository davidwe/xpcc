3.4 SCons Tools

In order make building our code with SCons as easy as possible we have created some tools which add special behaviour to SCons.

env = Environment(tools = ['xpcc'], toolpath = ['../../xpcc/scons'])

Since the tool xpcc needs other tools it will load them automatically, so the line above will load the following tools:

  • Tool: xpcc
  • Tool: configfile
  • Tool: helper
  • Tool: unittest
  • Tool: template
  • Tool: system_design

If the given architecture is atmega or atxmega than the following tools are also loaded:

  • Tool: avr
  • Tool: avrdude

If the given architecture is pc:

  • Tool: pc

If the given architecture is arm7tdmi, cortex_m0 or cortex_m3:

  • Tool: arm
  • Tool: openocd

This Tool is not loaded, because it is not needed for building code.

  • Tool: doxygen
