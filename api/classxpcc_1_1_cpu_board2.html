Detailed Description

template<typename Transmit = Empty, typename Receive = Empty>
class xpcc::CpuBoard2< Transmit, Receive >

Pin assignment:

CAN (MCP2515):

  • PC4 - CS
  • PC5 - SCK (SPIC)
  • PC6 - MISO (SPIC)
  • PC7 - MOSI (SPIC)
  • PB0 - INT
  • PB1 - TX_RTS0
  • PB2 - TX_RTS1
  • PB3 - TX_RTS2
  • PQ0 - RX_BF0
  • PQ1 - RX_BF1

RAM (23K256) & Flash (AT45DB041):

  • PQ2 - HOLD_RAM
  • PQ3 - CS_RAM
  • PF4 - CS_FLASH
  • PF5 - SCK (SPIF)
  • PF6 - MISO (SPIF)
  • PF7 - MOSI (SPIF)

BoardLink (to Slave):

  • PC0 - MS (output -> PD5)
  • PC1 - SM (input -> PD4)
  • PC2 - RXD (USARTC0)
  • PC3 - TXD (USARTC0)

Jtag:

  • PB4 - TMS
  • PB5 - TDI
  • PB6 - TCK
  • PB7 - TDO

Other:

  • PR0 - Button (active low, inverted via software)
  • PR1 - ext. clock (8 MHz)
  • PH0..7 - LEDs

Available on the headers:

  • PA0..7
  • PD0..7
  • PE0..7
  • PF0..7 (4-7 also used for the Flash)
  • PJ0..7
  • PK0..7
