static void xpcc::atxmega::TimerF0::setMsTimer ( uint8_t  interval = 1) [static]

Creates a medium level OVL interrupt every x ms. Sets the timer prescaler to 64 and calculates the timer period for the milliseconds specified. Interrupt handling has to be done manually. Accuracy is not guaranteed! The maximum interval is 131ms at 32MHz, and 2097ms at 2MHz.

Parameters:
intervalbetween interrupts in ms
