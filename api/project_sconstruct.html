4.1 Create a project using SCons

Two set up the project you need to create two files: SConstruct and project.cfg.

Just copy the example from the universal SConstruct file. How to create your project.cfg file is explained in detail on the page about configuration files.

Configuration file templates

Here are some templates for common targets:

PC

[general]
name = example

[build]
architecture = hosted

As you can see, this is the easiest case.

ATmega644

[general]
name = example

[build]
architecture = avr
device = atmega644
clock = 14745600

[avrdude]
port = usb
programmer = avrispmkII

[fusebits]
# ext. crystal oscillator, jtag disabled, brown-out level at 4.3V
lfuse = 0xef
hfuse = 0xd9
efuse = 0xfc

If you don't use the include avrdude facilities you can drop the avrdude and fusebit sections.

ATxmega128a1

[general]
name = example

[build]
architecture = avr
device = atxmega128a1
clock = 32000000

[avrdude]
port = usb
programmer = avrispmkII

[fusebits]
# Startup 4ms, JTAG enabled
fuse4 = 0xf6
# EESAVE, BODENABLE, BODLEVEL = 2.7V
fuse5 = 0xe2
