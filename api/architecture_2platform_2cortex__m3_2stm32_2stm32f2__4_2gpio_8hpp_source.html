00037 /* 
00038  * GPIO Ports
00039  * ==========
00040  * 
00041  * Each port bit of the General Purpose IO (GPIO) Ports, can be
00042  * individually configured by software in several modes:
00043  * - Input floating
00044  * - Input (with pull-up or pull-down)
00045  * - Analog
00046  * - Output open-drain (with pull-up or pull-down)
00047  * - Output push-pull (with pull-up or pull-down)
00048  * - Alternate function push-pull (with pull-up or pull-down)
00049  * - Alternate function open-drain (with pull-up or pull-down)
00050  * 
00051  * GPIOx_MODER (32-bit)
00052  *   00: Input
00053  *   01: General purpose output mode
00054  *   10: Alternate function mode
00055  *   11: Analog mode
00056  * GPIOx_OTYPER (16-bit)
00057  *   0: Output push-pull
00058  *   1: Output open-drain
00059  * GPIOx_OSPEEDR (32-bit)
00060  *   00: 2 MHz
00061  *   01: 25 MHz
00062  *   10: 50 MHz
00063  *   11: 100 MHz
00064  * GPIOx_PUPDR (32-bit)
00065  *   00: No pull-up, pull-down
00066  *   01: Pull-up
00067  *   10: Pull-down
00068  *   11: Reserved
00069  * 
00070  * GPIOx_IDR = input state (read-only) (16-bit)
00071  * GPIOx_ODR = output state (16-bit)
00072  * 
00073  * GPIOx_BSRR = set and reset register (32-bit)
00074  * - BSRR(i) sets
00075  * - BSRR(i+16) resets
00076  */
