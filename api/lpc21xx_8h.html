Detailed Description

Header file for all LPC23xx/24xx Microprocessors.

Header file for Philips LPC2114 / LPC2119 / LPC2124 / LPC2129 / LPC2194.

The header file is the super set of all hardware definition of the peripherals for the LPC23xx/24xx family microprocessor.
