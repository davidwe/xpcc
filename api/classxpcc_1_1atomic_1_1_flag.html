Detailed Description

This class is quite handy when exchanging flags between a interrupt routine and the main-loop.

 xpcc::atomic::Flag isrFlag;
 
 ISR() {
     ...
     isrFlag.set();
 }
 
 function() {
     bool flag = isrFlag.testAndSet(false);
     ...
 }
