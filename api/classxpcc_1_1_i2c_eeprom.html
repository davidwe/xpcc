Detailed Description

template<typename I2cMaster>
class xpcc::I2cEeprom< I2cMaster >

Compatible with the 24C256 (ST) and 24FC1025 (Microchip) family and other I2C eeprom with an 16-bit address pointer. Base address for most i2c eeproms is 0xA0.
