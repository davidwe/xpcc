Detailed Description

Collection of points, represented by their corresponding vectors. Used for example to hold the result of a intersection-operation.

Based on the xpcc::DynamicArray class, therefore grows automatically if more space than currently allocated is needed. But because this is an expensive operation it should be avoid if possible.
