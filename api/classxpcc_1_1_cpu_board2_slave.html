Detailed Description

template<typename Transmit = Empty, typename Receive = Empty>
class xpcc::CpuBoard2Slave< Transmit, Receive >

Pin assignment:

InterconnectLink (to Master):

  • PD4 - SM (output -> PC1)
  • PD5 - MS (input -> PC0)
  • PD6 - RXD (USARTD1)
  • PD7 - TXD (USARTD1)

Other:

  • PB0..3 - LEDs

On Headers:

  • PA0..7
  • PC0..7
  • PD0..3
