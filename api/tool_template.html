Tool: template

Provided builders and methods

Template(target, source, substitutions)

This is just a wrapper for the Python Template strings. substitutions needs to be Python dictionary with the all the placeholders use in the template file as keys. The values need to be simple strings.

substitutions = {
	'author' : 'Donald',
	'date' : today
}
target = env.Template(
		target = 'output/file.hpp'),
		source = 'input/file.hpp.in'),
		substitutions = substitutions)

See the official documentation for more information: Python - Template strings.

Jinja2Template(target, source, substitutions)

The Jinja2 template engine allows much more that simple text replacements inside the template file. It offers variables, functions, filter, conditional evaluation, loops etc.

As above substitutions needs to be Python dictionary. But now the values don't need to strings but can be Python objects, lists or dictionaries.

See the official documentation for more information: Jinja2 Documentation.
