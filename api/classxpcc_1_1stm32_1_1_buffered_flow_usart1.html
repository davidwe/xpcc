Detailed Description

Buffered UARAT with Hardware Flow Control.

This implementation uses a ringbuffer. The size of the ringbuffer can be changed in the project.cfg file.

Just add add a value for USART1_TX_BUFFER_SIZE and/or USART1_RX_BUFFER_SIZE in the defines section. The size can be any value between 1 and 254.

Example: 

 [defines]
 USART1_TX_BUFFER_SIZE = 20
 USART1_RX_BUFFER_SIZE = 30
