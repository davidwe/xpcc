Detailed Description

This implementation uses a ringbuffer. The size of the ringbuffer can be changed in the project.cfg file.

Just add add a value for USART6_TX_BUFFER_SIZE and/or USART6_RX_BUFFER_SIZE in the defines section. The size can be any value between 1 and 254.

Example: 

 [defines]
 USART6_TX_BUFFER_SIZE = 20
 USART6_RX_BUFFER_SIZE = 30
