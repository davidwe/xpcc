Detailed Description

The 12-bit ADC is a successive approximation analog-to-digital converter. It has up to 18 multiplexed channels allowing it measure signals from 16 external and two internal sources. The result of the ADC is stored in a left-aligned or right-aligned 16-bit data register.

This API is designed for the internal ADCs of STM32F4XX, STM32F10X_LD, STM32F10X_LD_VL, STM32F10X_MD, STM32F10X_MD_VL, STM32F10X_HD, STM32F10X_HD_VL, STM32F10X_XL and STM32F10X_CL.

Author:
Stephan Kugelmann, David Hebbeker

Enables the ADC clock and switches on the ADC. The ADC clock prescaler will be set as well.

The ADC clock must not exceed 14 MHz. Default is a prescaler of 8 which gives a 9 Mhz ADC clock at 72 MHz APB2 clock.

Parameters:
prescalerThe prescaler specifies by which factor the APB2 clock (72 MHz) will be divided.
