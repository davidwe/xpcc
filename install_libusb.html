avrdude und libusb

Ohne installierte libusb findet avrdude unter Windows keinerlei USB Programmer. Es erscheint nur eine Fehlermeldung wie did not find any USB device "usb".

Installation

Lade libusb-win32 herunter und entpacke es.

Schließe den Programmer an und starte den Wizard im bin Verzeichnis.

Er sollte darauf in der Liste der gefunden Geräte auftauchen.

Installiere den Treiber für den Programmer.

Windows beschwert sich das der Treiber nicht signiert ist, installiere ihn trotzdem. Danach muss der Programmer einmal abgezogen und neu anschloßen werden und avrdude sollte ihn erkennen.
